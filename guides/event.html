Event handling

Event handling

In Capsid, you can declare component's event handlers using @on decorator.

class MyButton {
  @on('click')
  onClick (e) {
    alert('clicked!')
  }
}

capsid.def('my-button', MyButton)

<button class="my-button">BUTTON</button>

See the working demo

@on('click') declares onClick method is the event handler of click event on el. Here, @ is decorators syntax of ESNext.

You need to set up babel for using babel-preset-decorators-legacy.

npm install --dev babel-preset-decorators-legacy

.babelrc:

{
  presets: [
    "es2016",
    "decorators-legacy",
  ]
}

or using babel-standalone with transform-decorators-legacy plugin:

<script src="https://unpkg.com/babel-standalone" data-plugins="transform-decorators-legacy">
// You can use decorators here
</script>
